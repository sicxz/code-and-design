## Everything Is Inside Something

- Almost every HTML element either contains other elements — or is contained by them
- This is **nesting**: how HTML creates structure and hierarchy
- Think of folders on your computer — a folder holds files and other folders, and those hold more
- Every element has a **parent** (the element it lives inside) and may have **children** (elements nested inside it)

Common parent-child relationships:
- A `<ul>` contains `<li>` elements
- A `<div>` contains paragraphs and headings
- A `<nav>` contains links

## The DOM Tree

- When a browser reads your HTML, it builds an internal model: the **Document Object Model (DOM)**
- The DOM is a tree — `<html>` is the root, `<head>` and `<body>` are its two main branches

```html
<html>
  <body>
    <section>
      <h1>Jazz Playlist</h1>
      <ul>
        <li>Kind of Blue — Miles Davis</li>
        <li>A Love Supreme — John Coltrane</li>
        <li>Maiden Voyage — Herbie Hancock</li>
      </ul>
    </section>
  </body>
</html>
```

That structure as a tree:
- `html`
  - `body`
    - `section`
      - `h1`
      - `ul`
        - `li` (×3)

Key vocabulary:
- **Child** — an element directly inside another element
- **Sibling** — an element at the same level, sharing the same parent
- Above: `<h1>` and `<ul>` are siblings — both are children of `<section>`

## Rules of Nesting

**Open before you close** — tags must nest properly; never close a parent before closing its children:

```html
<!-- Wrong: p closes before strong -->
<p>The concert was <strong>incredible</p></strong>

<!-- Right: strong closes inside p -->
<p>The concert was <strong>incredible</strong>.</p>
```

**Indent to show structure** — browsers ignore whitespace, but humans depend on it:
- Use 2 or 4 spaces consistently
- This is the single most important habit for readable HTML

```html
<!-- Hard to read — no indentation -->
<article><h2>The Record Store</h2><p>Vinyl only.</p><ul><li>Jazz</li><li>Soul</li></ul></article>

<!-- Easy to read — proper indentation -->
<article>
  <h2>The Record Store</h2>
  <p>Vinyl only. No streaming.</p>
  <ul>
    <li>Jazz</li>
    <li>Soul</li>
  </ul>
</article>
```

**Siblings example** — three `<a>` elements, all children of `<nav>`:

```html
<nav>
  <a href="/">Home</a>       <!-- child of nav, sibling of the other two -->
  <a href="/work">Work</a>   <!-- child of nav, sibling of the other two -->
  <a href="/about">About</a> <!-- child of nav, sibling of the other two -->
</nav>
```
