## 📋 Cheat Sheet (Quick Reference)

### SVG Element Quick Reference

```xml
<!-- Basic Structure -->
<svg width="24" height="24" viewBox="0 0 24 24">
  <!-- Shapes go here -->
</svg>

<!-- viewBox Syntax -->
viewBox="min-x min-y width height"
viewBox="0 0 24 24"  <!-- 24×24 coordinate system -->

<!-- Basic Shapes -->
<circle cx="12" cy="12" r="10" />           <!-- Center (12,12), radius 10 -->
<rect x="5" y="5" width="14" height="14" /> <!-- Rectangle -->
<line x1="0" y1="0" x2="24" y2="24" />      <!-- Diagonal line -->

<!-- Path Commands -->
M x y        <!-- Move to (doesn't draw) -->
L x y        <!-- Line to -->
H x          <!-- Horizontal line to x -->
V y          <!-- Vertical line to y -->
C x1 y1, x2 y2, x y  <!-- Cubic Bézier curve -->
Z            <!-- Close path (back to start) -->

<!-- Example Path -->
<path d="M5 2 L19 2 L19 22 L12 18 L5 22 Z" />

<!-- Groups -->
<g class="icon-group">
  <path d="..." />
  <path d="..." />
</g>

<!-- Accessibility -->
<svg role="img" aria-label="Bookmark icon">
  <!-- Interactive icon -->
</svg>

<svg aria-hidden="true">
  <!-- Decorative icon (with visible text) -->
</svg>
```

### Common Path Commands

| Command | Syntax | Description |
|---------|--------|-------------|
| `M` | `M x y` | Move to (pen up) |
| `L` | `L x y` | Line to |
| `H` | `H x` | Horizontal line to x |
| `V` | `V y` | Vertical line to y |
| `C` | `C x1 y1, x2 y2, x y` | Cubic Bézier curve |
| `S` | `S x2 y2, x y` | Smooth cubic Bézier |
| `Q` | `Q x1 y1, x y` | Quadratic Bézier |
| `A` | `A rx ry rotation large-arc sweep x y` | Arc |
| `Z` | `Z` | Close path |

### CSS Styling SVG

```css
/* Approach 1: CSS Variable */
.icon {
  --icon-color: var(--color-text);
}

.icon path {
  fill: var(--icon-color);
}

/* Approach 2: currentColor */
.icon {
  color: var(--color-text);
}

.icon path {
  fill: currentColor;  /* Inherits parent color */
}

/* Approach 3: Direct Fill */
.icon path {
  fill: var(--color-text);
}

/* Interactive States */
.icon-link:hover .icon path {
  fill: var(--color-accent);
}

.icon-link:focus-visible {
  outline: 2px solid var(--color-accent);
  outline-offset: 4px;
}
```

### ARIA Attributes for Icons

| Scenario | ARIA Attributes | Example |
|----------|-----------------|---------|
| Icon with visible text | `aria-hidden="true"` | Button with icon + "Save" text |
| Icon-only button | `role="img"` + `aria-label` | Icon button without text |
| Decorative icon | `aria-hidden="true"` | Icon for visual appeal only |
| Logo (link to home) | `role="img"` + `aria-label` | Header logo |

### Figma Export Checklist

**Before exporting:**
- [ ] Name layers semantically (`mark-outline`, not `Rectangle 7`)
- [ ] Flatten nested groups (or keep structure intentional)
- [ ] Outline strokes (Object → Outline Stroke)
- [ ] Use single color fills (CSS will control colors)
- [ ] Remove unnecessary effects (shadows, blur)
- [ ] Align to pixel grid (whole number coordinates)

**Export settings:**
- [ ] ☑ Outline stroke
- [ ] ☑ Flatten
- [ ] ☑ Simplify
- [ ] ☐ Include "id" attribute (add classes manually)

**After exporting:**
- [ ] Run through SVGO (SVG OMG)
- [ ] Manual cleanup (remove metadata, add classes)
- [ ] Add ARIA labels (if interactive)
- [ ] Test in browser (visual + keyboard + screen reader)
