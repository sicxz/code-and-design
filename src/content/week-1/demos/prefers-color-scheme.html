System Preference Detection

Zero JavaScript

This page adapts to your system's dark/light mode using only CSS. The @media (prefers-color-scheme: dark) query does all the work.

How It Works

The browser asks your operating system: "Does this user prefer dark mode?" Based on the answer, different CSS values are applied.

The Limitation

What if someone's system is set to dark mode, but they want YOUR site in light mode? CSS alone can't offer that override. That's where JavaScript comes in.

Try it yourself: Toggle your system's dark mode setting and watch this page update instantly. No refresh needed!

macOS: System Preferences → Appearance
Windows: Settings → Personalization → Colors
iOS: Settings → Display & Brightness
