CSS Variables Demo

One Source of Truth

Every blue element on this page uses --color-primary. Change it once in the CSS, and headings, buttons, links, and borders all update together.

Consistent Spacing

Padding, margins, and gaps all use the spacing scale. --space-sm, --space-md, --space-lg. Consistent rhythm throughout.

Try It Yourself

Open DevTools, find :root, and change --color-primary to something else. Watch the whole page respond.

Links use the same color — everything stays connected.

This is how design systems work. Tokens create consistency and make global changes effortless.
