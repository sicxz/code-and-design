The Pattern

JavaScript changes state. CSS responds to state. They stay in their lanes.

// 1. Find the toggle button
const toggle = document.querySelector('.theme-toggle');

// 2. Listen for clicks
toggle.addEventListener('click', function() {
  
  // 3. Toggle the 'dark' class on body
  document.body.classList.toggle('dark');
  
});

Open DevTools (Elements panel) and watch the <body> tag. Click the toggle and see the dark class appear and disappear.

One Source of Truth

Every color on this page comes from a CSS variable. Change one value, everything updates.

  • --color-surface → backgrounds
  • --color-text → all text
  • --color-accent → buttons & links
  • --color-border → all borders
