Overview

This assignment gets you set up with the professional tools you'll use all semester: GitHub, Figma, and CodePen. You'll create your workspace repository and document your setup for future reference.

Outcomes

By completing this assignment, you will:

  • Create your professional GitHub course repository
  • Set up GitHub account with proper profile
  • Create Figma account for design work
  • Set up CodePen for code experiments
  • Document your setup in your course repository README

Getting Started

Step 0: 🐙 GitHub Account

Go to: https://github.com/join

  • Use your @ewu.edu email address.
  • Choose a professional username (this becomes part of your portfolio URL).
  • Verify your email.

🐙 Set Up Your GitHub Profile

Click your avatar (top right) → Your profile

Click Edit profile and add:

  • Name: Your actual name (so I know who you are!)
  • Bio: Something like "DESN368 Student | Learning Web Design"
  • Location: Optional
  • Website: Optional (you'll have one soon!)

Click Save. Consider adding a profile photo (helps me learn names!)

Step 1: 🗂️ Workspace Setup

Go to the course repository: https://github.com/sicxz/desn368-s26-workspace-template

Click Fork (top right) → this creates your personal copy.

In your forked repo:

  • Edit README.md and update your My Toolkit section with links as you complete each tool setup.
  • Commit your changes.

Step 2: Follow Your Setup Checklist

Once your repository is created, you'll find a complete setup checklist at: setup/ONBOARDING-CHECKLIST.md

Step 3: Update Your Workspace

As you complete each tool setup, update your README.md file with:

  • Your profile links
  • Your learning goals
