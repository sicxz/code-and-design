Overview

Create a well-structured HTML recipe page using the recipe you brought to class. This assignment will help you practice the fundamental HTML elements we've covered in Week 1.

Objectives

  • Practice proper HTML document structure
  • Use semantic HTML elements appropriately
  • Understand the difference between ordered and unordered lists
  • Work with text formatting and emphasis
  • Add images and links to your page
  • Build confidence with CodePen as your development environment

Required HTML Elements

Your recipe page must include all of the following elements:

Headings

  • <h1> – Main recipe title
  • <h2> – At minimum, section headings (Examples: "Ingredients", "Instructions", "About This Recipe")

Text Elements

  • <p> – Paragraph text for descriptions, stories, or tips about your recipe
  • <br> – Line breaks where appropriate (use sparingly!)
  • <strong> – For important text that should be bold
  • <em> – For emphasized text (italics)

Lists

  • <ul> – Unordered list with minimum 3 items
    • Example use: Ingredients list
  • <ol> – Ordered list with minimum 3 items
    • Example use: Step-by-step instructions

Links

  • <a> – At least one hyperlink
    • Could link to ingredient information, cooking technique videos, or related recipes

Where to Work: CodePen

Getting Started with CodePen

  1. Go to codepen.io
  2. Click "Start Coding" or "Create" → "Pen"
  3. You'll see three panels: HTML, CSS, and JS
  4. For this assignment, you'll only work in the HTML panel
  5. The preview will update automatically as you type!

CodePen Tips

  • You don't need to include <!DOCTYPE html>, <html>, <head>, or <body> tags in CodePen – it handles those automatically
  • Start typing your <h1> tag right away!
  • Save your work frequently (click "Save" in the top right)
  • You can fork your pen to create versions if you want to experiment
