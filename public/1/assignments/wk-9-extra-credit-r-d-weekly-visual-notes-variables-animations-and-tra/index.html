WK EXTRA CREDIT – R&D // Weekly Visual Notes

Variables, Animations, and Transforms

Due: Thursday at 10:30 AM Required

Objective

Create hand-written visual notes that capture key concepts, ideas, and learnings from each week's content.

This Week's Focus

Read and take visual notes on these four articles about the journey ahead in web development.

Required Readings

Complete the tutorials and take notes along the way:

  1. Variables In CSS: The var() Function
  2. HTML video and audio
  3. CSS transitions and transforms for beginners
  4. An Interactive Guide to CSS Transitions
  5. An Interactive Guide to Keyframe Animations

Requirements

  • Pages: 4–6 pages per week (5.5" × 8.5" each)
  • Medium: Moleskine Cahier notebook
  • Tools: Pen (any color that scans well)
  • Style: Hand-written with visual elements (sketches, diagrams, connections, etc.)

What to Include

Document anything that helps you process and remember the week's material:

  • Key concepts and definitions
  • Quick sketches or diagrams
  • Questions that came up
  • Code snippets or structures
  • Connections between ideas
  • Personal insights or "aha" moments
  • Resources you want to remember

Submission

  1. Scan your pages using a document scanner or scanning app
  2. Create a single PDF file (right-reading order)
  3. Name your file: week-[#]-notes-lastname.pdf
  4. Upload to Canvas by Sunday at midnight

Tips

  • Don't worry about perfection – focus on capturing ideas
  • Mix text and visuals in whatever way helps you learn
  • Use your own note-taking style
  • These are YOUR learning artifacts

Grading

  • ✓ Complete (4–6 pages with genuine engagement)
  • ✓ Incomplete (less than 4 pages or minimal effort)
