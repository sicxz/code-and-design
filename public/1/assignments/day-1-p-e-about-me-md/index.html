Assignment Overview

Complete the Designer-Coder Intake form by editing the about-me.md file in your GitHub fork. This is your first markdown edit and GitHub commit!

Instructions

  1. Navigate to Your Fork
    • Go to YOUR repository (should be github.com/YOUR-USERNAME/desn368-s26-workspace-template)
    • Click into the week-1 folder
  2. Edit the about-me.md File
    • Click on about-me.md
    • Click the pencil icon (Edit this file)
  3. Fill Out the Form
    • Check boxes by adding an x between the brackets: [x]
    • Fill in text responses where indicated
    • Replace underscores with your answers
  4. Commit Your Changes
    • Scroll down to "Commit changes"
    • Add commit message: "Complete designer-coder intake"
    • Click "Commit changes"

Submission

Submit the link to your edited about-me.md file:

https://github.com/YOUR-USERNAME/desn368-s26-workspace-template/blob/main/week-1/about-me.md

Tips

  • To check a box in markdown: change [ ] to [x]
  • Preview your changes before committing
  • Make sure you're editing in YOUR fork, not the original repo
