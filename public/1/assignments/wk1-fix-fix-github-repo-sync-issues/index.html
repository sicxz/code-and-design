Fix Github Repo Sync Issues

Code & Design Workspace · Spring 2026

Clone and Import Repo

  1. Make a brand-new repo that isn't a fork
  2. Go to GitHub (logged in).
  3. Click the + (top-right) → Import repository.
  4. Old repository URL: paste the URL of your current repo (the one that says "forked from …").
  5. Repository name: DESN368-code-design-workspace-INITIALS-new-copy
  6. Keep it Public
  7. Click Begin import and wait. When it's done, open the new repo. It should not say "forked from …".

Cleanup

Once you confirm the new repo works

  1. Go back to the old forked repo:
  2. Click Settings → General → Danger Zone → Delete this repository
Submission Required

Submit the link to your new repo to Canvas.
