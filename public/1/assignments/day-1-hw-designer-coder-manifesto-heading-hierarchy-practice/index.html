Assignment Overview

Read the content at code-and-design.org/1 and then write a short Designer-Coder Manifesto in HTML.

Your manifesto should include exactly one <h1>, one each of <h2> through <h6>, and one paragraph under each heading.

Resource

Start here:

https://code-and-design.org/1

What You Need to Do

  1. Read the course content at code-and-design.org/1
  2. Write a short Designer-Coder Manifesto
  3. Use one heading at each level from <h1> to <h6>
  4. Write one paragraph under each heading
  5. Keep the heading levels in the correct order
  6. If you missed class on Tuesday or need help go here: https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Structuring_content/Headings_and_paragraphs

Submission Instructions

  1. Work in CodePen
    • Open CodePen.io and create a new Pen
    • Write or paste your HTML into the HTML panel
    • Do not worry about CSS yet
  2. Save Your Work
    • Name your Pen: h1-manifesto-lastname-firstname-s26
    • Save the Pen
  3. Submit Here
    • Copy your CodePen URL
    • Paste it into the submission box below
    • Click Submit

Checklist Before Submitting

  • ✓Contains exactly one <h1> with one paragraph
  • ✓Contains one each of <h2> through <h6> with one paragraph each
  • ✓No skipped heading levels
  • ✓Pen is named correctly
  • ✓Pen is saved and viewable
Note

Keep this simple. This is a day one HTML structure exercise. We are focusing on heading hierarchy, not styling.
