Console Playground Week 0

A page built for experimentation. Open DevTools and start exploring.

Getting Started

Open your browser's console and try these commands:

document.querySelector('h1')
document.querySelector('h1').textContent = 'Hello!'
document.querySelector('h1').style.color = 'red'

Press F12 or Cmd+Option+I (Mac) / Ctrl+Shift+I (Windows) to open DevTools.

Things to Try

  • Find elements: document.querySelector('.color-box')
  • Change styles: document.querySelector('.color-box').style.borderRadius = '50%'
  • Find all buttons: document.querySelectorAll('button')
  • Count list items: document.querySelectorAll('li').length
  • Find the hidden message and make it visible
